# ProbotG - AI Powered Chatbot

### Overview

ProBotG is a multi-tasking chatbot. It performs tasks like sending an e-mail, scheduling a meeting, performing topic- based searches, text summarization, retrieving weather information, etc.

This chatbot is powered by Llama 3.3 70B Versatile, which is a high-performance, multilingual, and instruction-tuned LLM from Meta.

### Tech Stack

* Python
* Streamlit
* Langchain
* Groq API

### Environment (IDE)

1. VS Code
2. PyCharm
3. Datastax Astra
